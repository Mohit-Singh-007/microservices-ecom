# Microservices E-Commerce Backend

## Tech Stack
- Spring Boot
- Spring Cloud
- Eureka Service Discovery
- API Gateway
- REST APIs
- Docker (optional)

## Services
- User Service
- Product Service
- Order Service
- API Gateway
- Service Registry (Eureka)

## Architecture
Microservices architecture with service discovery and gateway routing.

## Run Locally
1. Start Eureka Server
2. Start Services
3. Start API Gateway
